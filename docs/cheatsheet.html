Linux Cheat Sheet

Linux Command Description
ls -1 | wc -l Count number of files in directory
ls -p | grep -v / Shows list of only the files in the current directory
grep "[word/phrase]" [file to search] > [file to write to] Search a file for a specific word or phrase and save all instances of this to a new file


Slurm Command Description
sacct --starttime [YYYY]-[MM]-[DD] Slurm job history for your account for a specified day
sinfo -o "%60N %10P %8t %8D %10X %10Y %10m %25G %b " | grep [partition name] Provides detailed description of partition: availability, number of sockets and CPUs, type of node, etc.
